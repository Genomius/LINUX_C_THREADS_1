## gcc main.c -o main -std=c99 -lpthread -g && ./main
	* -std=c99 - C99-стандарт Си. Флаг был добавлен для использования конструкции for(int i, , );
	* -lpthread - Использование потоков в программе
	* -g - Включен режим отладки

## Отладка gdb
	* $ gdb main
	* (gdb) run
